运算符

加法运算符

加法运算符可完成两种运算：数值间的加法、字符串间的连接。注：加号不能用于操作不同类型的两个值，如 数值 + 字符串。

算术运算符

算术运算符：加（+）、减（-）、乘（*）、除（/）、整除（//）、取余（%）、幂（**）、正数值运算符（+a）、负数值运算符（-a）。

位运算符

位运算符有如下这些：与（&）、或（|）、否（~）、异或（^）、左移（<<）、右移（>>）。

赋值运算符

赋值运算符有如下这些：= 、+= 、-+ 、*= 、**= 、/= 、//= 、%= 、>>= 、<<= 、&= 、|= 、^= 。

比较运算符

比较运算符有如下这些：== 、!= 、> 、>= 、< 、<= 。== 只判断值是否相等，而不判断对象地址。Python 支持链式比较：

1 < 2 < 3

逻辑运算符

逻辑运算符有如下这些：且（and）、或（or）、非（not）。

且（and）、或（or）返回的都是运算子的值，非（not）返回的是布尔值：True、False 。

其他运算符

成员运算符

成员运算符有：in 、not in 。用于判断某个值是否在序列中，返回 True 或 False 。

身份运算符

身份运算符有：is 、is not 。用于比较两个对象的存储单元。可用 id() 函数获取对象的内存地址。

a = [1, 2, 3]
b = a[:]
b is a  #False
b == a  #True
