函数

定义函数

使用 def 关键字来定义一个函数，函数可以有多个返回值，默认返回None。

def add(x,y):
    return x + y

参数

Python 中的参数可以分为多种：位置参数、关键字参数、默认值参数、不定长参数 。

一、位置参数

调用函数时 位置参数 必须按顺序传入，并且数量必须与申明时的一样。

def person(name, age):
    print('%s is %d years old'%(name, age))
person('God', 10000)

二、关键字参数

关键字参数 是指调用函数时明确的指明参数名。此时不要求参数的传入顺序，只要保证函数能正确的获取到参数。注：关键字参数必须在位置参数之后传入，同一个参数不能重复传入。

def person(sex, name, age):
    print('the %s %s is %d years old'%(sex, name, age))
person(age=10000, name='God', sex='male')
person('male', age=10000, name='God')

三、默认值参数

声明函数时，可以给参数设置默认值。调用时若没有传入相应的参数，则使用其默认值。注：默认值参数 必须在位置参数之后声明。

def person(sex, name, age = 20):
    print('the %s %s is %d years old'%(sex, name, age))
person('male', 'God', 10000)
person('male', 'God')
person(name='God', sex='male')

参数默认值的创建和绑定是发生在函数定义的时候，以后每次调用默认参数时都会使用同一个引用。

def f(x = []):
    x.append(1)
#调用3次f()后，默认值列表将变为[1, 1, 1]

四、不定长参数

不定长参数 指参数数目不定，可用星号(*)来构造。注：不定长参数 必须在位置参数之后声明。

def add(x, *args):
    total = x
    for arg in args:
        total += arg
    print(total)
add(1, 2, 3)

*args 表示不定长参数，可以看成一个元组。调用这种形式的函数时，不能使用 关键字传参。

def add(x, **kwargs):
    total = x
    for arg, value in kwargs.items():
        total += value
    print(total)
add(10, y=11, z=12, w=13)
add(y=11, z=12, w=13, x=10)

**kwargs 表示不定长(关键字)参数，相当于一个字典。

def log(*args, **kwargs):
    print(args, kwargs)
log(1, 2, a=3, b=4)

此函数可接收任意数目的位置参数和关键字参数。

参数使用 元组、字典 的方式传入

def add(x, y):
    print(x + y)
tup = (1, 2)
add(*tup)

dic = {'y':2, 'x':1 }
add(**dic)

解构赋值

a, b, c = [1, 2, 3]
a, b, c = (1, 2, 3)
a, b, c = {1, 2, 3}

列表、元组、集合 都可使用这种方法解构赋值，字典不行。注：左侧变量的数量必须与右侧的元素数量相同。

return语句

使用 return 返回值并退出函数，函数默认返回None，不带参数值的return语句(return;)也返回None。函数可返回多个值，相当于返回一个元组。

def log(a, b):
    return a, b
x, y = log(1, 2)
tup = log(1, 2)
