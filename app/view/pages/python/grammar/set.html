集合

简介

集合 是一种无序的不重复的序列。为了保证元素的唯一性，元素只能是不可变的。元素不能是列表、字典、集合等。

st = {1,2,3}
st = set([1,2,3])
st = { x for x in range(10) }

不可变集合

相对于集合（set），可以使用 frozenset() 来创建一个不可变集合，不可变集合不能添加和删除元素。一个主要应用是作为字典的键。

st = frozenset([1, 2, 3, 'a', 1])

集合操作符

操作符 说明
| 取并集，返回包含两个集合所有元素的集合（去除重复）。a | b 同 a.union(b)
& 取交集，返回包含两个集合共有元素的集合。a & b 同 a.intersection(b)
- 取差集，返回只在 a 不在 b 的元素组成的集合。a - b 同 a.difference(b)
^ 取对称差集，返回在 a 或在 b 中，但是不同时在 a 和 b 中的元素组成的集合。a ^ b 同 a.symmetric_difference(b)
<= 、< 判断是否是子集或真子集。a <= b 同 a.issubset(b)
>= 、> 判断是否是超集。a >= b 同 a.issuperset(b)
for in 遍历集合。for s in {1,2,3}: print(s)
in、not in 成员运算符，判断集合中是否包含给定的元素。'a' in {'a', 1} 返回True

方法

方法 说明
add() 添加元素，如果已存在则无变化。s.add('abc')
update() 添加多个元素。s.update(seq) 将seq中的元素添加到s中。s.update([5, 6, 7])
remove() 移除元素，若元素不存在则报错。s.remove(ob)
discard() 移除元素，但元素不存在时不报错。s.discard(ob)
pop() 随机删除一个元素，并返回。空集合调用此方法则报错。
difference_update() 从a中去除所有属于b的元素。a.difference_update(b)
