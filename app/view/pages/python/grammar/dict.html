字典

简介

字典（dict）是一种由键值对组成的无序数据结构。在一些编程语言中也被称为 hash 或 map 。

键必须是唯一的，且不可变的。键可以是：数字、布尔值、字符串、元组，不推荐使用浮点数。重复的键以后定义的为准。

dic = {'name': 'yang', 'age': 20}
dic = dict([('name','yang'),('age',20)])
dic = { x:y for x,y in zip([1,2,3], ['a','b','c']) }

字典操作符

操作符 说明
[key] 通过 key 获取值，可读可写。访问不存在的key则报错。赋值时，若存在则修改，不存在则新增。
for in 遍历字典。for key in dic: print(key)
in、not in 成员运算符，判断字典中是否包含给定的key。'a' in {'a': 1} 返回True
del 删除字典中的键。del dic['a']。删除不存在的键则报错

类方法

类方法 说明
fromkeys() 创建一个新字典。dict.fromkeys(seq, value=None)。以序列seq中的各个元素做为键，value为值，默认为None

实例方法

实例方法 说明
items() 返回由键值组成的类元组列表，可遍历。dict_items([(key1,value1),(key2,value2)])
keys() 返回由所有的键组成的类列表，可遍历。dict_keys([key1,key2])
values() 返回由所有的值组成的类列表，可遍历。dict_values([value1,value2])。可用list()转为列表
get() 返回指定键的值，如果键不存在则返回默认值 None。dict.get(key, default=None)
setdefault() 为字典添加键值，若已存在则无变化，若不存在则设为默认值。dict.setdefault(key, default=None)
update() 以一个字典为参数，批量修改字典。dict.update(dict2) 已存在的则修改，不存在的则添加
copy() 返回一个字典的浅复制。
clear() 清空字典的所有元素。
pop() 删除指定的键值。若存在则返回删除的值，若不存在则返回默认值，此时必须传默认值，否则报错。pop(key[,default])
popitem() 随机删除一对键值并返回该键值组成的元组，空字典调用此方法则报错。
