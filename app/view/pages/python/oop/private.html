私有属性和方法

简介

在 Java 和 C++ 中，可以用 private 和 protected 关键字修饰属性和方法，它们控制属性和方法能否被外部或者子类访问。在 Python 中约定以双下划线（__）为前缀的属性和方法名为私有的，拒绝在类外进行访问。注：不要以双下划线结尾。

class MyClass(object):
    def __private(self):
        print("I'm private!")

mc = MyClass()
mc.__private()  #将报错
mc._MyClass__private()  #可以访问

这种私有属性和方法可以通过 obj._Classname__privateAttributeOrMethod 来进行访问。同时以双下划线结尾的属性和方法不行。

特殊属性和方法

Python 中有一些同时以双下划线为前缀和后缀的特殊属性和方法。

特殊属性和方法 说明
__class__ 获取实例对象的类型对象
__name__ 获取某个类型对象的名字
__new__() 构造函数，创建实例对象时调用
__init__() 初始化函数，初始化实例对象时调用
__del__() 析构函数，释放对象时使用
__len__() 获得长度
__str__() 对象转字符串，使用 print() 函数时显示该结果
__repr__() 对象转字符串，非使用 print() 函数时显示该结果
__setitem__() 按照索引赋值
__getitem__() 按照索引获取值
__call__() 函数调用
__cmp__() 比较运算
__add__(self,other) 加运算
__sub__() 减运算
__mul__() 乘运算
__div__() 除运算
__mod__() 求余运算
__pow__() 乘方运算
