Mock.js 根据一定的规则生成数据，用于前后端分离开发。官方文档 官方示例

Mock.js 定义了特定的语法规范，包括两部分：

  1. 数据模板定义规范（Data Template Definition，DTD）
  2. 数据占位符定义规范（Data Placeholder Definition，DPD）

Mock.js 提供了一些 API ：Mock.mock()、Mock.setup()、Mock.valid()、Mock.toJSONSchema() 和 Mock.Random 工具类。
