Mock.valid()

简介

Mock.valid( template, data )

校验真实数据 data 是否与数据模板 template 匹配。template 可以是对象或字符串，例如 { 'list|1-10':[{}] }、'@EMAIL'。

注：经过测试，只能判断字符串是否相等，无法判断是否匹配。

var template = {
    name: 'value1'
}
var data = {
    name: 'value2'
}
Mock.valid(template, data)
// =>
[
    {
        "path": [
            "data",
            "name"
        ],
        "type": "value",
        "actual": "value2",
        "expected": "value1",
        "action": "equal to",
        "message": "[VALUE] Expect ROOT.name'value is equal to value1, but is value2"
    }
]
