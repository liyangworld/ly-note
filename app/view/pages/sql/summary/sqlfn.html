SQL函数速览手册

简介

SQL 拥有很多用于计数和计算的内建函数。

Aggregate 函数

Aggregate (聚合)函数用于计算从列中取得的值，返回一个单一的值。

常用函数 说明
AVG() 返回数值列的平均值。SELECT AVG(column_name) FROM table_name
COUNT() 返回行数。SELECT COUNT(column_name) FROM table_name;（NULL 不计入）。SELECT COUNT(DISTINCT column_name) FROM table_name;（返回指定列有不同值的数目）
FIRST() 返回第一个记录的值。SELECT FIRST(column_name) FROM table_name;只有 MS Access 支持该函数
LAST() 返回最后一个记录的值。SELECT LAST(column_name) FROM table_name;只有 MS Access 支持该函数
MAX() 返回指定列的最大值。SELECT MAX(column_name) FROM table_name;
MIN() 返回指定列的最小值。SELECT MIN(column_name) FROM table_name;
SUM() 返回数值列的总和。SELECT SUM(column_name) FROM table_name;

Scalar 函数

Scalar 函数基于输入值，返回一个单一的值。

常用函数 说明
UCASE() 将列值转换为大写。SELECT UCASE(column_name) FROM table_name; SQL Server使用UPPER()
LCASE() 将列值转换为小写。SELECT LCASE(column_name) FROM table_name; SQL Server使用LOWER()
MID() 从某个文本字段提取字符，MySql 中使用。SELECT MID(column_name,start[,length]) FROM table_name;注：start从1开始
SubString(字段，1，end) 从某个文本字段提取字符
LEN() 返回某个文本字段中 值的长度。SELECT LEN(column_name) FROM table_name; MySQL使用LENGTH()
ROUND() 对某个数值字段进行指定小数位数的四舍五入。SELECT ROUND(column_name,decimals) FROM table_name; 其中decimals为小数点后位数，默认为0
NOW() 返回当前的系统日期和时间。SELECT NOW() as _date FROM table_name;
FORMAT() 格式化某个字段的显示方式。SELECT FORMAT(column_name,format) FROM table_name;
