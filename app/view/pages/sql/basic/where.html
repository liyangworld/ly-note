where

简介

WHERE 子句用于过滤行。

SELECT column_name,column_name
FROM table_name
WHERE column_name operator value;

WHERE 子句中的运算符

值为字符串时需用单引号括起来，值为数值时不带引号。

运算符 说明
= 等于
<> 不等于。注：在 SQL 的一些版本中，该操作符可被写成 !=
> 大于
< 小于
>= 大于等于
<= 小于等于
BETWEEN 在某个范围内（MySQL包含临界值）。where column_name between 1500 and 3000;值可以是数值、文本或者日期。相应地有 not between
IN 多个可能值之一。where column_name in (5000,3000,1500);
LIKE 模糊查询。where column_name like 'xxx%'; 相应地有 not like
is null 空值判断。where column_name is null;相应地有 is not null

WHERE 子句中的逻辑运算符

逻辑运算符 说明
And 与，同时满足。where sal > 2000 and sal < 3000;
Or 或，满足其中之一。where sal > 2000 or comm > 500;
Not 非，满足不符合该条件的。where not sal > 1500;

LIKE 中使用的通配符

通配符 说明
% 替代 0 个或多个字符。如：'https%'（以https开头）、'%oo%'（包含oo）
_ 替代一个字符
[abc] 字符列中的任何单一字符
[^abc]或[!abc] 不在字符列中的任何单一字符

MySQL 中可以使用 REGEXP 或 NOT REGEXP 运算符 (或 RLIKE 和 NOT RLIKE) 来操作正则表达式。如：WHERE name REGEXP '^[A-H]'; name以A-H开头
