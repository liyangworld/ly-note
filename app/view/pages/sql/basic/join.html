join

简介

JOIN 用于把来自两个或多个表的行进行合并。JOIN 有四种类型。

JOIN 类型 说明
INNER JOIN 对表中的每一个匹配返回一行，无匹配则忽略
LEFT JOIN 以左表为基准，即使左表中的行在右表中无匹配，也返回一行，该行中属于右表的字段以空值填充
RIGHT JOIN 以右表为基准，即使右表中的行在左表中无匹配，也返回一行，该行中属于左表的字段以空值填充
FULL JOIN 对表中的每一个匹配返回一行，无匹配的行也返回一行，该行中属于其他表的字段以空值填充

一、INNER JOIN

SELECT column_name(s)
FROM table1
INNER JOIN table2
ON table1.column_name=table2.column_name;

SELECT column_name(s)
FROM table1
JOIN table2
ON table1.column_name=table2.column_name;

注：INNER JOIN 与 JOIN 是相同的。

二、LEFT JOIN

SELECT column_name(s)
FROM table1
LEFT JOIN table2
ON table1.column_name=table2.column_name;

注：在某些数据库中，LEFT JOIN 被称为 LEFT OUTER JOIN。

三、RIGHT JOIN

SELECT column_name(s)
FROM table1
RIGHT JOIN table2
ON table1.column_name=table2.column_name;

注：在某些数据库中，RIGHT JOIN 被称为 RIGHT OUTER JOIN。

四、FULL JOIN

SELECT column_name(s)
FROM table1
FULL JOIN table2
ON table1.column_name=table2.column_name;

注：在某些数据库中，FULL JOIN 被称为 FULL OUTER JOIN。MySQL 不支持 FULL JOIN，可以使用UNION ALL子句 将 左、右连接合并。

SELECT column_name(s)
FROM table1
LEFT JOIN table2
ON table1.column_name=table2.column_name;
UNION ALL
SELECT column_name(s)
FROM table1
RIGHT JOIN table2
ON table1.column_name=table2.column_name;
