Number对象

Number对象是三个包装对象之一，用来生成数值的包装对象。

1. var num = 123 ;
2. var num = Number("123") ;
3. var num = new Number(123) ;

属性

静态属性 说明
prototype 对象的原型，可为对象新增实例属性和方法。Number.prototype.new=null;
NaN 表示非数值，指向NaN
MAX_VALUE 表示最大的正数
MIN_VALUE 表示最小的正数
POSITIVE_INFINITY 正的无限，指向Infinity
NEGATIVE_INFINITY 负的无限，指向-Infinity
EPSILON 表示 1 和大于 1 的最小值的差值
MAX_SAFE_INTEGER 表示能够精确表示的最大整数，即9007199254740991
MIN_SAFE_INTEGER 表示能够精确表示的最小整数，即-9007199254740991

方法

静态方法 说明
parseInt(str,radix) 根据给定的进制数把一个字符串解析成整数，与全局的 parseInt() 相同
parseFloat 把一个字符串解析成浮点数，与全局的 parseFloat() 相同
isNaN 判断给定的参数是否为NaN，区别于全局的isNaN()，该方法不会将参数强制转换为数值
isFinite 检测传入的参数是否是一个有穷数，区别于全局的isFinite()，该方法不会将参数强制转换为数值，
isInteger 判断给定的参数是否为整数，NaN 和正负 Infinity 不是整数
isSafeInteger 判断传入的参数值是否是一个“安全整数”，[-(253 - 1),253 - 1]
实例方法 说明
toExponential(n) 以指数表示法返回该数值字符串表示形式，n表示小数位数（四舍五入）
toFixed(n) 把数字转换为字符串并返回，n限定小数点后的位数（四舍五入）
toPrecision(n) 把数字转换为字符串并返回，n限定数字的位数（四舍五入），用于四舍五入时不太可靠

继承的属性、方法

继承的属性、方法 说明
toString(radix) 覆盖了Object 对象的 toString 方法，radix默认为10
toLocaleString() 返回这个数字在特定语言环境下的表示字符串
valueOf() 返回一个Number对象的原始值
