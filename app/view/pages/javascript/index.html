JavaScript 的兼容性处理

IE9 及以上对 es5 支持很好。因此 js 的兼容性处理方案如下：

  1. 只需支持IE9及以上：用Babel转译到 es5 即可。
  2. 需支持IE8及以下：先用Babel转译到 es5，然后引用es5-shim.js填补一些数组等方法，再引入json3.js填补JSON.parse和JSON.stringify方法。

需注意Object.defineProperty无法填补，因此Vue无法支持IE8。
