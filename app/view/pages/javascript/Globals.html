全局变量、方法

全局变量

全局变量 说明
NaN 代表非数字值
Infinity 代表正无穷大,（负无穷大为 -Infinity）
undefined 代表未定义的值

全局方法

全局方法 说明
isNaN() 判断给定的参数是否为NaN，区别于Number.isNaN()，该方法会将参数强制转换为数值
isFinite() 检测传入的参数是否是一个有穷数，区别于Number.isFinite()，该方法会将参数强制转换为数值，
parseInt(str,radix) 根据给定的进制数把一个字符串解析成整数，与Number.parseInt() 相同
parseFloat 把一个字符串解析成浮点数，与Number.parseFloat() 相同
eval(str) 计算 JavaScript 字符串，并把它作为脚本代码来执行
escape(str) 对字符串进行编码，除了ASCII的字母、数字和括号中的字符（ * @ - _ + . /），其他所有的字符都会被转义序列替换
unescape(str) 对由 escape() 编码的字符串进行解码
encodeURI(str) 把字符串编码为 URI
decodeURI(str) 解码某个编码的 URI
encodeURIComponent(str) 把字符串编码为 URI组件
decodeURIComponent(str) 解码某个编码的 URI组件
