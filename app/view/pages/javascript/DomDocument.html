document对象

简介

document对象指文档的根节点，每张网页都有自己的document节点。只要浏览器开始载入HTML文档，这个节点对象就存在了，可以直接调用。

获取document节点的几种方法：

  1. 对于正常的网页，直接使用document或window.document
  2. 对于iframe载入的网页，使用iframe节点的contentDocument属性
  3. 对Ajax操作返回的文档，使用XMLHttpRequest对象的responseXML属性
  4. 对于包含某个节点的文档，使用该节点的ownerDocument属性

属性

属性 说明
defaultView 在浏览器中返回document对象所在的window对象，否则返回null
doctype 返回文档类型节点
documentElement 返回当前文档的根元素（即html）
head 返回文档的head元素
body 返回文档的body元素
activeElement 返回当前文档中获得焦点的那个元素。用户通常可以使用Tab键移动焦点，使用空格键激活焦点。
forms 返回当前文档中所有表单元素form
images 返回页面所有图片元素（即img标签）
links 返回当前文档所有设定了href属性的a及area元素
embeds 返回网页中所有嵌入对象，即embed标签
scripts 返回当前文档的所有脚本（即script标签）
styleSheets 返回当前网页的所有样式表
documentURI 返回当前文档的网址，可用于所有文档（包括 XML 文档）
URL 返回当前文档的网址，只能用于 HTML 文档
location 返回location对象
domain 返回当前文档的域名
lastModified 返回当前文档最后修改的时间戳，格式为字符串
referrer 返回一个字符串，表示当前文档的访问来源，总是与HTTP头信息的Referer保持一致
title 返回当前文档的标题
characterSet 返回渲染当前文档的字符集，比如UTF-8、ISO-8859-1
readyState 返回当前文档的状态：loading(加载HTML代码阶段,尚未完成解析)、interactive(加载外部资源阶段时)、complete(加载完成时)
designMode 控制当前文档是否可编辑，可选值 "on"、"off"，类似的有el.contentEditable属性
implementation 返回一个对象，用来甄别当前环境部署了哪些DOM相关接口。该属性的hasFeature方法，可以判断当前环境是否部署了特定版本的特定接口
compatMode 返回浏览器处理文档的模式，可能的值为BackCompat（向后兼容模式）和CSS1Compat（严格模式）。如果网页代码的第一行设置了明确的DOCTYPE（比如），document.compatMode的值都为CSS1Compat
cookie 用来操作浏览器Cookie

方法

方法 说明
querySelector() 返回文档中匹配指定的CSS选择器的第一个元素。IE8支持CSS2选择器
querySelectorAll() 返回文档中匹配指定的CSS选择器的所有元素列表，一个NodeList对象。IE8支持CSS2选择器
getElementById() 独有方法，返回文档中指定id的元素
getElementsByName() 返回文档中所有指定name的元素集合，一个NodeList对象
getElementsByClassName() 返回文档中所有指定类名的元素集合，一个HTMLCollection对象
getElementsByTagName() 返回文档中所有指定标签名的元素集合，一个HTMLCollection对象
elementFromPoint(x,y) 返回位于页面指定位置最上层的Element子节点
createElement() 用来生成网页元素节点，如果参数里面包含尖括号（即<和>）会报错
createDocumentFragment() 生成一个DocumentFragment对象，该对象是一个存在于内存的DOM片段，但是不属于当前文档
createTextNode() 用来生成文本节点
createAttribute() 生成一个新的属性对象节点，参数为属性名，用value设置值
createComment() 创建一个注释节点
importNode(node,deep)() 把一个节点从另一个文档复制到该文档以便应用，deep为true则一并复制子孙节点
adoptNode() 将某个节点，从其原来所在的文档移除，插入当前文档，并返回插入后的新节点
createEvent() 生成一个事件对象，该对象可以被element.dispatchEvent方法使用，触发指定事件
addEventListener() 向文档添加事件句柄（IE8及之前使用attachEvent()）
removeEventListener() 移除文档中的事件句柄（IE8及之前使用detachEvent()）
dispatchEvent() 触发指定事件
open() 用于新建一个文档，供write方法写入内容。它实际上等于清除当前文档，重新写入内容
close() 用于关闭open方法所新建的文档。一旦关闭，write方法就无法写入内容了
write() 用于向当前文档写入内容
writeln() 同write()方法，但会在输出内容的尾部添加换行符
hasFocus() 返回一个布尔值，表示当前文档之中是否有元素被激活或获得焦点
getSelection() 同window.getSelection()
