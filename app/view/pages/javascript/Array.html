Array对象

Array 作为构造函数使用时，不同的参数，会导致它的行为不一致：无参数时，返回一个空数组；多参数时，所有参数都是新数组的成员；单参数时，非数值则是新数组成员，正整数则是新数组的长度，非正整数则报错。new 关键字可省略。

1、var arr = ["dog","cat","fish"];
2、var arr = new Array("dog","cat","fish");
3、var arr = new Array(3); // arr.length === 3

属性

静态属性 说明
prototype 对象的原型，可为对象新增实例属性和方法。Array.prototype.new=null;
实例属性 说明
length 返回数组的长度，可读可写

方法

以下方法中改变原数组的有：push、pop、unshift、shift、splice、reverse、sort、copyWithin 。

静态方法 说明
isArray 确定参数是否是一个 Array，优于instanceof
of 创建一个数组
from 从一个类似数组或可迭代的对象中创建一个新的数组实例
实例方法 说明
push 向数组的末尾添加一个或多个元素，并返回数组改变后的长度
pop 删除数组的最后一个元素并返回删除的元素
unshift 向数组的开头添加一个或多个元素，并返回数组改变后的长度
shift 删除数组的第一个元素并返回删除的元素
splice(i,n,"item") 在数组i位置处，向后删除n个元素(并添加多个元素)，返回由删除元素组成的数组
slice(i1,i2) 获取子数组并返回，含头不含尾。i1、i2可为负值，-1最后元素，-2倒数第二，以此类推。i1所指位置始终在i2前面
concat 将多个数组中的元素合并成一个新数组并返回
join 把数组的所有元素用分隔符连接成一个字符串并返回，(无參时用逗号连接)
reverse 反转数组的元素顺序，并返回改变后的数组
sort 对数组的元素进行排序并返回，默认按字母升序，可传入排序函数作为参数，如：function(a,b){return a-b;}
forEach(fn(curV,i,arr), thisV) 遍历数组
indexOf("item",start) 搜索数组中的元素并返回其(从start开始)第一次出现的位置索引
lastIndexOf("item",start) 从后往前查询位置索引，返回从start开始向前首次出现的位置索引
reduce(fn,iValue) 对累加器和数组中的每个元素 (从左到右)应用一个函数，将其减少为单个值并返回
reduceRight(fn,iValue) 对累加器和数组中的每个元素 (从右到左)应用一个函数，将其减少为单个值并返回
map(fn) 返回一个由 经过处理后的元素 所组成的新数组。参数fn：function(currentItem,index,arr){return ?}
filter(fn) 返回一个由 符合条件的元素 所组成的新数组。参数fn：function(currentItem,index,arr){return ?}
every(fn) 检测数组所有元素是否都符合指定条件,是则返回true。空数组返回true
some(fn) 检测数组中是否有元素符合指定条件,有则返回true。空数组返回false
find(fn) 返回数组中满足提供的测试函数的第一个元素的值，无则返回 undefined
findIndex(fn) 返回数组中满足提供的测试函数的第一个元素的索引，无则返回-1
keys 返回一个新的Array迭代器，它包含数组中每个索引的键
values 返回一个新的 Array Iterator 对象，该对象包含数组每个索引的值
entries 返回一个新的Array Iterator对象，该对象包含数组中每个索引的键/值对
fill(value,start,end) 用一个固定值填充一个数组中从起始索引到终止索引内的全部元素
copyWithin(target,start,end) 浅复制数组的一部分到同一数组中的另一个位置，并返回它，而不修改其大小
includes(item,start) 判断一个数组是否包含一个指定的值，返回true / false

继承的属性、方法

继承的属性、方法 说明
toString() 返回对象的字符串形式。覆盖了Object 对象的 toString 方法
toLocaleString() 返回一个字符串，数组中的元素将使用各自的 toLocaleString 方法转成字符串
valueOf() 返回一个Array对象的原始值。返回值为原数组
