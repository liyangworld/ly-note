Date对象

Date对象 用来处理日期和时间。可以表示的时间范围是，1970年1月1日00:00:00前后的各1亿天。

1、var d1 = new Date() ;  //当前日期和时间
2、var d6 = new Date(ms) ;  //ms指距1970年1月1日之间相差的毫秒数
3、var d2 = new Date("month dd,yyyy hh:mm:ss") ;  //month指英文月份，其他为数字
4、var d3 = new Date("month dd,yyyy") ;
5、var d4 = new Date(yyyy,mth,dd,hh,mm,ss) ;  //皆为数字，mth指月份索引值( 0-11 )
6、var d5 = new Date(yyyy,mth,dd) ;

属性

静态属性 说明
prototype 对象的原型，可为对象新增实例属性和方法。Date.prototype.new=null;

方法

Date对象的方法主要分为2种形式：本地时间和UTC时间。同一个方法，一般都会有此2种时间格式操作。UTC时间( Universal Coordinated Time )即世界标准时间，与GMT(格林尼治时间)相同。北京时间的时区为东8区，比GMT时间早8个小时。

静态方法 说明
parse() 解析日期字符串，并返回毫秒数
UTC() 接受的参数同日期构造函数接受最多参数时一样，返回毫秒数
now() 返回自1970年1月1日 00:00:00 UTC到当前时间的毫秒数
本地时间 get/set方法 说明
getFullYear() 从 Date 对象以四位数字返回年份
getMonth() 从 Date 对象返回月份 (0 - 11)
getDate() 从 Date 对象返回一个月中的某一天 (1 - 31)
getHours() 返回 Date 对象的小时字段 (0 - 23)
getMinutes() 返回 Date 对象的分钟字段 (0 - 59)
getSeconds() 返回 Date 对象的秒数字段 (0 - 59)
getMilliSeconds() 返回 Date 对象的毫秒字段(0 - 999)
getDay() 从 Date 对象返回星期(0 - 6)，0为星期天
getTime() 返回 1970 年 1 月 1 日至今的毫秒数
getTimezoneOffset() 返回GMT 时间和本地时间之间的时差，以分钟为单位（如：-480）
setFullYear(yyyy,mth,dd) 设置年份，第一个参数必填
setMonth(mth,dd) 设置月份，第一个参数必填
setDate(dd) 设置一个月的某一天
setHours(hh,mm,sec,msec) 设置小时，第一个参数必填
setMinutes(mm,sec,msec) 设置分钟，第一个参数必填
setSeconds(sec,msec) 设置秒，第一个参数必填
setMilliSeconds(msec) 设置毫秒
setTime(msec) 以毫秒设置 Date 对象
其他方法 说明
toLocaleString() 可根据本地时间把 Date 对象转换为字符串并返回，结果格式如下：2016/11/29 下午4:57:11
toUTCString() 根据世界时 (UTC) 把 Date 对象转换为字符串并返回，结果格式如下：Tue, 29 Nov 2016 09:30:16 GMT
toDateString() 可把 Date 对象的日期部分转换为字符串并返回，结果格式如下：Tue Nov 29 2016
toLocaleDateString() 根据本地时间把 Date 对象的日期部分转换为字符串并返回，结果格式如下：2016/11/29
toTimeString() 可把 Date 对象的时间部分转换为字符串并返回，结果格式如下：17:04:52 GMT+0800 (中国标准时间)
toLocaleTimeString() 根据本地时间把 Date 对象的时间部分转换为字符串并返回，结果格式如下：下午5:06:43
toISOString() 可以使用ISO标准（UTC时间）将 Date 对象转换为字符串并返回，结果格式如下：2016-11-29T08:58:57.717Z
toJSON() 将 Date 对象转换为字符串，并格式化为 JSON 数据格式，结果格式如下：2016-11-29T08:58:57.717Z （同ISO）
UTC时间 get/set方法 说明
getUTCFullYear() 根据世界时从 Date 对象以四位数字返回年份
getUTCMonth() 根据世界时从 Date 对象返回月份 (0 - 11)
getUTCDate() 根据世界时从 Date 对象返回一个月中的某一天 (1 - 31)
getUTCHours() 根据世界时返回 Date 对象的小时字段 (0 - 23)
getUTCMinutes() 根据世界时返回 Date 对象的分钟字段 (0 - 59)
getUTCSeconds() 根据世界时返回 Date 对象的秒数字段 (0 - 59)
getUTCMilliSeconds() 根据世界时返回 Date 对象的毫秒字段(0 - 999)
getUTCDay() 根据世界时从 Date 对象返回星期(0 - 6)，0为星期天
setUTCFullYear(yyyy,mth,dd) 根据世界时设置年份，第一个参数必填
setUTCMonth(mth,dd) 根据世界时设置月份，第一个参数必填
setUTCDate(dd) 根据世界时设置一个月的某一天
setUTCHours(hh,mm,sec,msec) 根据世界时设置小时，第一个参数必填
setUTCMinutes(mm,sec,msec) 根据世界时设置分钟，第一个参数必填
setUTCSeconds(sec,msec) 根据世界时设置秒，第一个参数必填
setUTCMilliSeconds(msec) 根据世界时设置毫秒

继承的属性、方法

继承的属性、方法 说明
toString() 覆盖了Object 对象的 toString 方法。返回 Tue Nov 29 2016 16:53:55 GMT+0800 (中国标准时间)
valueOf() 返回一个Date对象的原始值。即毫秒值
