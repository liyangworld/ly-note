navigator/screen 对象

navigator 对象

属性

属性 说明
navigator.platform 返回运行浏览器的操作系统平台,如：Win32
navigator.appName 返回浏览器的名称。IE是“Microsoft Internet Explorer”； 其他浏览器是“Netscape”。
navigator.appCodeName 返回浏览器的代码名。所有浏览器都返回“Mozilla”
navigator.appVersion 返回浏览器的平台和版本信息。如 Chrome ： 5.0 (Windows NT 6.1; WOW64) AppleWebKit/537.36 (KHTML, like Gecko) Chrome/54.0.2840.71 Safari/537.36
navigator.userAgent 返回浏览器用于 HTTP 请求的用户代理头的值。如 Chrome ： Mozilla/5.0 (Windows NT 6.1; WOW64) AppleWebKit/537.36 (KHTML, like Gecko) Chrome/54.0.2840.71 Safari/537.36
navigator.cookieEnabled 返回布尔值，判断浏览器是否启用了 cookie
navigator.onLine 返回布尔值，判断浏览器是否连接网络了
navigator.geolocation 返回一个Geolocation对象，包含用户地理位置的信息

screen 对象

属性

属性 说明
screen.width、screen.height 返回设备屏幕的宽度及高度（单位px）
screen.availwidth、screen.availheight 返回设备屏幕的可用宽度及高度（单位px），如不包括系统任务栏
screen.colorDepth 返回设备或缓冲器上调色板的比特深度（比特每像素），如：24
screen.pixelDepth 返回显示屏幕的颜色分辨率（比特每像素），同上，如：24
