Object对象

JavaScript 原生提供Object对象，所有其他对象都继承自这个对象。Object本身也是一个构造函数，可以直接通过它来生成新对象。

var obj = {a: 1};
var obj = new Object({a: 1});

属性

静态属性 说明
prototype 对象的原型，可为对象新增实例属性和方法。Object.prototype.new=null;

方法

静态方法 说明
create 使用指定的原型对象及其属性去创建一个新的对象
keys 返回一个数组，由对象的自身可枚举属性组成
getOwnPropertyNames 返回一个数组，由对象 自身拥有的枚举或不可枚举属性名称字符串组成
defineProperty 在一个对象上定义一个新属性，或者修改一个对象的现有属性，并返回这个对象
defineProperties 在一个对象上定义新的属性或修改现有属性，并返回该对象
seal 让一个对象密封，并返回被密封后的对象
isSealed 判断一个对象是否被密封
freeze 冻结一个对象并返回，冻结之后该对象不允许任何修改
isFrozen 判断一个对象是否被冻结
preventExtensions 让一个对象变的不可扩展，也就是永远不能再添加新的属性
isExtensible 判断对象是否是可扩展的（是否可以在它上面添加新的属性）,默认情况下对象是可扩展的
getPrototypeOf 返回对象的原型。如果没有，则返回 null
getOwnPropertyDescriptor 返回对象上一个自身属性对应的属性描述符
assign 将所有可枚举的属性的值从源对象复制到目标对象并返回，改变原对象、浅拷贝
is 判断两个参数是否是同一个值
getOwnPropertySymbols 返回一个数组，由对象自身的所有 symbol 属性键组成
setPrototypeOf 设置对象的原型
values 返回一个数组，由对象自身可枚举的属性值组成
entries 返回一个由对象自身可枚举属性[key，value]对组成的数组
getOwnPropertyDescriptors 返回对象的所有自身属性的描述符，如果没有任何自身属性，则返回空对象
实例方法 说明
hasOwnProperty 判断参数是否是对象的自身（不继承）属性
isPrototypeOf 判断对象是否存在于参数的原型链上
propertyIsEnumerable 判断指定的属性名是否是当前对象的可枚举的自身属性
toString 返回一个表示该对象的字符串
toLocaleString 返回一个表示该对象的字符串
valueOf 返回指定对象的原始值
