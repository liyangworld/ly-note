RegExp对象

正则表达式（regular expression）是描述字符模式的对象，是对字符串执行模式匹配的强大工具。

以正则表达式为参数的 String对象 方法有 search()、match()、replace()、split() 。

1、var reg = new RegExp(模式,修饰符) ;  如：var reg = new RegExp("\\w+");  //这里用到了字符转义( \ )
2、var reg = /模式/修饰符 ;        如：var reg = /\w+/;

修饰符

修饰符 说明
i 忽略大小写
g 全局匹配（查找所有匹配的，而非找到第一个匹配就停止）
m 多行匹配

方括号

表达式 说明
[abc] 查找方括号之间的任何字符（a、b、c）
[^abc] 查找任何不在方括号之间的字符（非 a、b、c）
[0-9] 查找任何从0到9的数字
[a-z] 查找任何从小a到小z的字母
[A-Z] 查找任何从大A到大Z的字母
[A-z] 查找任何从大A到小z的字母和符号（包括所有字母及之间的符号）
red|green|blue 查找任何指定的选项（red 或 green 或 blue）

量词

量词 说明
n？ 包含零个或一个n
n*、n*? 包含零个或多个n，依次为贪婪模式、非贪婪模式
n+、n+? 包含一个或多个n，依次为贪婪模式、非贪婪模式
n{X} 包含X个n（X个连续重复的n）
n{X,} 包含至少X个n
n{X,Y} 包含 [X,Y] 个n
^n 以n为开头
n$ 以n为结尾
m(?=n) 匹配其后紧接n的m
m(?!n) 匹配其后没有紧接n的m

元字符

元字符 说明
. 代表任何单个字符，除了换行和行结束符
\w、\W 依次代表单词字符、非单词字符
\d、\D 依次代表数字、非数字
\s、\S 依次代表空白字符、非空白字符
\b、\B 依次代表单词边界、非单词边界（可用在开头或结尾）
\n、\r、\t、\v、\f 依次代表换行符、回车符、制表符、垂直制表符、换页符
\cX 表示Ctrl-[X]，其中的X是A-Z之中任一个英文字母
\0 代表null字符（U+0000）
\xxx 代表以八进制数xxx表示的字符
\xdd 代表以十六进制数dd定义的字符，如十六进制 57 (W)
\uxxxx 代表以十六进制数xxxx定义的Unicode字符

属性

实例属性 说明
ignoreCase 返回一个布尔值，表示是否设置了i修饰符，该属性只读
global 返回一个布尔值，表示是否设置了g修饰符，该属性只读
multiline 返回一个布尔值，表示是否设置了m修饰符，该属性只读
source 返回正则表达式的字符串形式（不包括反斜杠），该属性只读
lastIndex 返回下一次开始搜索的位置。该属性可读写，但是只在设置了g修饰符时有意义

方法

实例方法 说明
reg.test(str) 检索字符串str是否有匹配的值，有则返回true，无则返回false
reg.exec(str) 检索字符串str，如果发现匹配，就返回一个数组，成员是每一个匹配成功的子字符串，否则返回null

继承的属性、方法

继承的属性、方法 说明
toString() 返回对象的字符串形式。覆盖了Object 对象的 toString 方法
valueOf() 返回一个RegExp对象的原始值。返回值同 toString 方法一样
