String对象

String对象是三个包装对象之一，用来生成字符串的包装对象。

1. var str = "123" ;
2. var str = String(123) ;
3. var str = new String("123") ;

属性

静态属性 说明
prototype 对象的原型，可为对象新增实例属性和方法。String.prototype.new=null;
实例属性 说明
length 返回字符串的长度，只可读

方法

以正则表达式为参数的方法有 search()、match()、replace()、split() 。

静态方法 说明
fromCharCode 使用给定的一系列Unicode值创建字符串。但是无法处理高位编码的字符
fromCodePoint 同fromCharCode，区别是此方法可以处理高位编码的字符
raw 唯一一个内置的模板字符串标签函数，返回模板字符串的原始字面量值
实例方法 说明
indexOf(str,start) 返回指定的字符(串)在该字符串中(从start开始)首次出现的位置索引，无则返回-1
lastIndexOf(str,start) 从后往前查询位置索引，返回截止start之前首次出现的位置索引
search 查找子串或与正则匹配的值，并返回首次出现的索引值或-1，类似于indexOf()
match 在字符串中检索指定的字符(串)或多个正则匹配，返回数组或null
replace 在字符串中查找匹配的子串，并将其替换成其他子串，返回新字符串
split 将字符串分割成数组并返回。()和("未匹配")都将得到单元素数组，("")将把每个字符分割开。第二个参数限定数组长度
slice(i1,i2) 提取子串并返回，取头不取尾。
substr(i,n) 提取从指定索引i开始的n个字符串并返回。
substring(i1,i2) 提取子串并返回，取头不取尾。i1、i2不分先后。
concat 连接多个字符串，并返回新的字符串
toLowerCase 把字符串转为小写
toUpperCase 把字符串转为大写
toLocaleLowerCase 把字符串转为小写
toLocaleUpperCase 把字符串转为大写
charAt(i) 返回在指定位置的字符
charCodeAt(i) 返回在指定位置的字符的Unicode编码
localeCompare 比较两字符串的排序，返回负值（前）、0、正值（后）
trim 去除字符串两边的空白
codePointAt(i) 同charCodeAt，区别是该方法可处理高位编码
includes(str,start) 判断一个字符串是否包含另外一个字符串，start默认为0
endsWith(str,end) 判断当前字符串是否是以给定的子字符串“结尾”的，返回true/false，end索引不算，默认为原字符串长度
startsWith(str,start) 判断当前字符串是否是以给定的子字符串“开头”的，返回true/false，start索引算，默认为0
repeat(n) 重复字符串n次并返回新字符串
normalize 按照指定的一种 Unicode 正规形式将当前字符串正规化，默认值为"NFC"，可选值有："NFC", "NFD", "NFKC","NFKD"
padStart(n,str) 填充字符串头部直到字符串长度为n
padEnd(n,str) 填充字符串尾部直到字符串长度为n

继承的属性、方法

继承的属性、方法 说明
toString() 返回对象的字符串形式。覆盖了Object 对象的 toString 方法
valueOf() 返回一个String对象的原始值。返回值同 toString 方法一样
