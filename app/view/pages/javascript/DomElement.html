element 对象

简介

element 对象对应网页的HTML标签元素。每一个HTML标签元素，在DOM树上都会转化成一个Element节点对象。element 对象 具有Node对象、ParentNode接口、ChildNode接口的所有属性和方法。

属性

属性 说明
attributes 返回一个类数组对象(动态的)，成员是当前元素节点的所有属性节点
id 返回指定元素的id属性，可读写
tagName 返回指定元素的大写标签名，与nodeName属性的值相等
title 返回指定元素的title属性，可读写
className 返回当前元素节点的class属性，可读写
classList 返回一个类数组对象，该对象有以下方法：add()、remove()、toggle()、contains()、item()、toString()
style 返回元素的style属性，可读写
innerHTML 返回该元素包含的 HTML 代码，可读写。如果插入的是文本，最好用textContent
outerHTML 返回一个字符串，内容为指定元素节点的所有HTML代码，包括它自身和包含的所有子元素。可读写
previousElementSibling 返回元素的前一个兄弟元素，无则返回null
nextElementSibling 返回元素的后一个兄弟元素，无则返回null
offsetParent 返回元素最靠近的、position属性不等于static的上层元素。若当前元素display为none、或position为fixed，则返回null
clientLeft、clientTop 返回元素左、上边框的宽度（border），若有滚动条则包括滚动条的宽度
clientWidth、clientHeight 返回元素节点可见部分的宽、高，不含边框及以外部分
offsetWidth、offsetHeight 返回元素的宽、高，包含Border以及滚动条，不包含溢出部分
offsetLeft、offsetTop 返回元素左上角相对于Element.offsetParent节点的水平位移、垂直位移
scrollWidth、scrollHeight 返回元素的总宽、高，包含溢出部分，但不包含border和margin
scrollLeft、scrollTop 返回元素此时滚动的距离，可读写

方法

方法 说明
querySelector() 返回第一个匹配的子元素，该方法先在全局搜索选择器，然后过滤出该元素的子元素
querySelectorAll() 返回所有匹配的子元素，该方法先在全局搜索选择器，然后过滤出该元素的子元素
getElementsByTagName() 返回当前元素的所有指定标签子元素集合，一个HTMLCollection对象
getElementsByClassName() 返回当前元素的所有指定类名子元素集合，一个HTMLCollection对象
closest(selector) 返回当前元素的最接近的父元素(或自身)
match(selector) 判断元素是否匹配给定的选择器
getAttribute() 获取某个属性的值
setAttribute() 设置某个属性
hasAttribute() 判断是否有某个属性
removeAttribute() 移除某个属性
hasAttributes() 判断元素是否有属性
getAttributeNode() 获取某个属性节点
setAttributeNode() 设置某个属性节点
addEventListener() 向元素添加事件句柄（IE8及之前使用attachEvent()）
removeEventListener() 移除元素中的事件句柄（IE8及之前使用detachEvent()）
dispatchEvent() 触发指定事件
insertAdjacentHTML(position,htmlStr) 解析HTML字符串，然后将生成的节点插入到指定位置。position有四个值：beforebegin(在当前元素前面)、afterbegin(在当前元素里面，插在所有子元素之前)、beforeend(在当前元素里面，插在所有子元素之后)、afterend(在当前元素后面)
getBoundingClientRect() 返回一个对象，该对象有以下属性：width、height、left、top、right、bottom。都是以视口左上角为起点，border计算在内
getClientRects() 返回一个类数组对象，每个成员都同getBoundingClientRect()方法返回的对象一样。块级元素则返回一个成员，行内元素则根据所占行数返回多个成员；而getBoundingClientRect()永远只返回一个
focus() 使当前元素获得焦点
scrollIntoView(true) 滚动当前元素进入浏览器的可见区域，类似于设置window.location.hash的效果。true则顶部对齐，false则底部对齐
