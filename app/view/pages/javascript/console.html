console对象

console对象是 JavaScript 的原生对象，可以输出各种信息到控制台。

console.log 使用方法

该方法可以接受多个参数，将它们的结果连接起来输出。

支持以下占位符：%s (字符串)、%d (整数)、%i (整数)、%f (浮点数)、%o (对象的链接)、%c (CSS格式字符串)。

console.log('a', 'b', 'c'); // a b c
console.log(' %d + %d = %d ', 1, 1, 2,'end'); //  1 + 1 = 2 end
console.log('%cThis text is styled!','color: red; background: yellow; font-size: 24px;');

自定义方法

['log', 'info', 'warn', 'error'].forEach(function(method) {
    console[method] = console[method].bind(console,new Date().toISOString());
});
console.log("出错了！"); // 2017-09-04T07:03:12.871Z 出错了！

console 方法

静态方法 说明
log 在控制台输出信息
info 同log
debug 同log
warn 同log，但会在最前面加一个黄色三角，表示警告
error 同log，但会在最前面加一个红色的叉，表示错误
table 将复合类型的数据 {}、[{},{}] 转为表格显示
count 用于计数被调用了多少次
dir、dirxml 以易于阅读和打印的格式显示 {}
assert 进行条件判断，如果不满足条件，就显示一个错误，但不会中断程序执行
time、timeEnd 用于计时，可以算出中间操作所花费的准确时间
profile、profileEnd 开启、结束性能测试器（profile）
group、groupEnd、groupCollapsed 将显示的信息分组折叠
trace 显示当前执行的代码在堆栈中的调用路径
clear 清除控制台

命令行 API

属性、方法 说明
$_ $_属性返回上一个表达式的值
$0 - $4 控制台保存了最近5个在Elements面板选中的DOM元素，$0代表倒数第一个，$1代表倒数第二个
$(selector)、$$(selector) 类似于document.querySelector()、querySelectorAll()
$x(path) 返回一个数组，包含匹配特定XPath表达式的所有DOM元素
inspect(el) 打开相关面板，并选中相应的元素
getEventListeners(el) 返回一个{}，键为各种事件，值为回调函数组成的数组
keys(obj)、values(obj) 类似于Object.keys()、Object.values()
monitorEvents(el[, events]) 监听特定对象上发生的特定事件
unmonitorEvents(el[, events]) 取消监听
profile([name])，profileEnd([name]) 启动、结束一个特定名称的CPU性能测试
clear() 清除控制台
copy(el) 复制特定DOM元素到剪贴板
dir(obj)、dirxml(obj) 是console.dir、console.dirxml 方法的别名

debugger语句

debugger语句主要用于除错，作用是设置断点。

for(var i = 0; i < 5; i++){
    console.log(i);
    if (i === 2) debugger;
}
