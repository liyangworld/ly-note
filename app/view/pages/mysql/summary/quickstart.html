快速安装使用

安装

可以在官网上下载各操作系统对应的社区版本进行安装。

将会安装 MySQL服务器 和 MySQL 客户端程序等。安装好之后可以将bin目录添加到环境变量中，以便使用 mysqladmin、mysql 等命令。

启动MySQL服务器

MySQL数据库服务器通常指的的是mysqld，而命令行mysql则是mysql客户端程序。启动mysql服务器即是启动mysqld进程，mysqld启动后，可以通过mysql连接到mysql服务器。

1、使用mysqld 命令启动，后面可接各种选项：

mysqld --console --print-defaults // 启动MySQL服务器，并打印默认配置

2、mysqladmin 关闭服务器

mysqladmin -uroot -pxxx shutdown

打开MySQL客户端

mysql -u root -p

回车之后输入密码，即可进入。

若是第一次打开，则默认的root用户密码为空，此时直接回车即可。但是不能执行任何操作，MySQL服务器要求必须设置密码再执行其它操作。使用以下命令创建root用户密码：

mysqladmin -u root password "new_password"

进入之后便可进行各种操作，如查看服务器中所有的数据库：

show databases;
use xxx; //切换到某个数据库

图形界面工具 Navicat

在实际使用中可以安装图形界面工具 Navicat来方便对数据库进行操作。Navicat官网

如果后续忘记了服务器密码，则可以在Navicat上很方便的更改密码。
