querystring - 查询字符串

简介

querystring 模块主要是提供了一些实用函数，用于解析与格式化 URL 的查询字符串。

const querystring = require('querystring');

方法

方法 说明
querystring.parse(str[, sep[, eq[, options]]]) 把一个URL的查询字符串 str 解析成一个键值对的集合。sep默认为'&'，eq默认为'='。该方法返回的对象不继承自JavaScript的 Object 类，因此obj.toString()、obj.hasOwnProperty() 等没有被定义且无法使用。
querystring.stringify(obj[, sep[, eq[, options]]]) 通过遍历给定的 obj 对象的自身属性，生成 URL 查询字符串。sep默认为'&'，eq默认为'='。如果obj对象的属性值类型不是string、number、boolean 或 这三者组成的数组，则被强制转换为空字符串。
querystring.escape(str) 该方法是提供给 querystring.stringify() 使用的，通常不直接使用。它之所以对外开放，是为了在需要时可以通过给 querystring.escape 赋值一个函数来重写编码的实现。
querystring.unescape(str) 对给定的 str 进行解码。该方法是提供给 querystring.parse() 使用的，通常不直接使用。 它之所以对外开放，是为了在需要时可以通过给 querystring.unescape 赋值一个函数来重写解码的实现。
