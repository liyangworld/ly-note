简介

2009年5月 Ryan Dahl 在 GitHub 上发布了Node.js的初版。

作者创造Node.js的目的是为了实现高性能Web服务器，首先看重的是事件机制和异步IO模型的优越性。

Node.js 是一个基于 Chrome V8 引擎的 JavaScript 运行环境。
